Arduino Lie Detector

Interactive hardware • Arduino • GSR + pulse • LEDs + sound

Overview A playful “lie detector” built with Arduino that reads galvanic skin response (GSR) and pulse to infer arousal changes, driving LEDs and a buzzer for immediate feedback.
My Role Circuit design, Arduino code, enclosure, user testing
Sensors GSR (skin conductance), pulse sensor
Outputs LED bargraph, piezo buzzer, serial logging

Circuit & Prototype

How It Works

The GSR sensor measures small changes in skin conductance correlated with arousal. Readings are smoothed and mapped to an LED bargraph and buzzer. The pulse sensor adds a heartbeat cue for timing. This is not a medical device or a real polygraph—it's an interactive demo showing how bio-signal inputs can drive playful outputs.

Demo Video

Notes & Learnings

Sensor signals are noisy—simple moving averages helped. Grounding, stable power, and consistent finger contact made a big difference. Calibrating per-user avoids false positives. The project is intentionally lighthearted but sparked great conversations about biometrics and interpretation.
