Arduino Lie Detector

Interactive hardware • Arduino • GSR + pulse • LEDs + sound

Overview

A playful “lie detector” built with Arduino that uses a galvanic skin response (GSR) sensor and a pulse sensor. Readings are mapped to LEDs and sound, creating an interactive demo of how bio-signals can be turned into playful outputs.

Notes & Learnings

Biometric signals are noisy — I used simple moving averages to stabilize readings. Calibration is important since each person’s baseline differs, which helps avoid false positives. This project was a fun way to explore the intersection of hardware, design, and playful interaction.
